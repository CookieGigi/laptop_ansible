# Presets

Pre-configured group collections for common setups.

## Overview

Presets bundle multiple groups together for specific use cases.

## Available Presets

- **workstation** - Development workstation setup
- **gaming-rig** - Gaming optimized configuration
- **laptop** - Mobile power-efficient setup
- **server** - Minimal server configuration
- **minimal** - Bare essentials only

## Quick Reference

- **Structure**: See [structure.md](structure.md)
- **Apply**: `make preset NAME=<preset>`
- **Location**: `presets/*.yml`
