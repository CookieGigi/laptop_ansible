# Alacritty

## Overview

Alacritty is a modern terminal emulator that uses GPU acceleration. This role installs and configures Alacritty.

## Installation

- **Method**: APT
- **Role Path**: [`/ansible/roles/terminal/alacritty`](file:///home/cookiegigi/Project/system_maintain/ansible/roles/terminal/alacritty)

## Configuration

- **Installation**: Installs Alacritty using APT.
- **Configuration**: Configures Alacritty using the `alacritty_config` role.

## Usage

Alacritty is used as a terminal emulator with GPU acceleration.

No specific variables are required for this role.
