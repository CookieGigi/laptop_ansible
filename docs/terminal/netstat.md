# Netstat

## Overview

Netstat is a command-line network utility tool that displays network connections for Transmission Control Protocol (both
incoming and outgoing), routing tables, and a number of network interface and network protocol statistics. This role
installs Netstat using Nix.

## Installation

- **Method**: Nix
- **Role Path**: [`/ansible/roles/terminal/netstat`](file:///home/cookiegigi/Project/system_maintain/ansible/roles/terminal/netstat)

## Configuration

- **Installation**: Installs Netstat using the `nix_install` role.

## Usage

Netstat is used to monitor network connections and statistics.

No specific variables are required for this role.
