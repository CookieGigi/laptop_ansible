# basedpyright

basedpyright is a Python language server protocol (LSP) implementation based on Pyright. It provides type checking, code completion, and other IDE features for Python development.

## Installation

```bash
cd ansible && ansible-playbook playbooks/dev.yml --tags basedpyright
```

## What it does

- Installs basedpyright via npm
- Provides LSP capabilities for Python in text editors
- Offers fast and accurate type checking
- Supports all Python versions
